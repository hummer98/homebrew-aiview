# homebrew-aiview

Homebrew Cask tap for [AIview](https://github.com/hummer98/AIview).

## Install

```bash
brew tap hummer98/aiview
brew install --cask aiview
```

## Update

```bash
brew upgrade --cask aiview
```

The cask is automatically updated by a GitHub Actions workflow when a new release is published on [AIview](https://github.com/hummer98/AIview/releases).
